# Portafolio - Didier Najas

Portafolio personal desplegado en GitHub Pages.

## Estructura del proyecto

    Basic-portafolio/
    ├── index.html       # Página principal (punto de entrada para GitHub Pages)
    ├── estilos.css      # Estilos globales (variables, secciones, tema claro/oscuro)
    ├── script.js        # Lógica frontend (tema, idioma, animaciones, formulario)
    └── img/
        ├── Didier-Najas-photo.jpg   # Foto de perfil
        └── portfolio.png            # Imagen de referencia

## Tecnologías

Frontend:  HTML5, CSS3, JavaScript, Bootstrap 5
Backend:   Node.js, Java, Spring Boot, Python (proyectos externos)
BD:        MySQL, PostgreSQL, SQL
Librerías: Font Awesome, Google Fonts (Poppins, JetBrains Mono), EmailJS

## Funcionalidades

- Tema oscuro / claro con persistencia en localStorage
- Cambio de idioma ES / EN
- Animación de contadores en la sección de estadísticas
- Formulario de contacto vía EmailJS
- Modales de detalle por proyecto
- Scroll reveal con IntersectionObserver
- Navbar activa según sección visible
- 100% responsivo

## GitHub Pages

El sitio está configurado para servirse desde la raíz del repositorio.
El archivo index.html es el punto de entrada.

URL: https://didiernajas.github.io/Portafolio-Didier-Najas/
